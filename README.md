# SGPA Calculator

A web-based SGPA calculator for engineering students.  
It calculates SGPA based on subject marks and credit distribution for different departments.

## Features
- Supports multiple departments
- Calculates SGPA automatically
- Clean and simple UI
- Input validation for marks
- Responsive layout

## Departments Included
- Computer Engineering (CE)
- Information Technology (IT)
- Electronics & Telecommunication (ENTC)
- Artificial Intelligence & Data Science (AIDS)
- Electronics & Computer Engineering (ECE)

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Use
1. Select your department
2. Enter marks for each subject
3. Click **Submit**
4. Your SGPA will be displayed instantly
